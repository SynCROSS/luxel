GitHub Issues in this repo; use `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles map 1:1 to GitHub labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context: read `CONTEXT-MAP.md`, then relevant `CONTEXT.md` per area; system ADRs in `docs/adr/`. See `docs/agents/domain.md`.
